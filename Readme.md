# Fullstack Kanban Board
A Kanban board application. [Link for the deployment.](https://07prajwal2000kanbanboard.vercel.app/)

Technologies used:
- NodeJs
- ExpressJs
- NextJs
- MongoDB
- ReactJs
- TailwindCSS
- ShadCN

Basically a **MERN** but with a NextJs.
